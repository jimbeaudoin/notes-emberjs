# notes-emberjs
Ember.js Notes for Myself

### Installation via ember-cli
```sh
npm install -g ember-cli
npm install -g bower
npm install -g phantomjs
```
### New Application Creation
```sh
ember new my-new-app
cd my-new-app
ember serve
```
